Sobre MERKATU

Transformando o varejo com tecnologia

Nossa Missão

MERKATU é uma plataforma inovadora que conecta lojas físicas e virtuais, oferecendo aos clientes uma experiência integrada de compra online com retirada em loja ou entrega. Com foco em gerenciamento eficiente de estoque e geolocalização das lojas, facilitamos o comércio varejista moderno.

Recursos Principais

  • ✓ Catálogo de produtos integrado entre todas as lojas
  • ✓ Gerenciamento centralizado de estoque em tempo real
  • ✓ Localização de lojas com mapa interativo
  • ✓ Painel administrativo completo
  • ✓ Busca inteligente de produtos por loja
  • ✓ Relatórios e analíticas de vendas

Tecnologia

MERKATU utiliza tecnologias modernas e escaláveis:

Frontend: Vite + JavaScript
Backend: Node.js + Express
Mapas: Leaflet.js + OpenStreetMap
API: RESTful com JSON
